# Proyecto_c
## Descripción 
Este proyecto es una herramienta de terminal escrita en lenguaje C diseñada para realizar conversiones de divisas en tiempo real entre Quetzales (GTQ), Dólares (USD) y Euros (EUR). Al ser un lenguaje de bajo nivel y compilado, el programa destaca por su eficiencia y manejo directo de tipos de datos numéricos para cálculos financieros precisos.
## Prerrequisitos (tecnologías a instalar)
Compilador de C: Es necesario tener instalado GCC (GNU Compiler Collection), Clang o un entorno como MinGW en Windows.
Terminal: Acceso a la línea de comandos (CMD, PowerShell, Bash o terminal de VS Code).
## Pasos para su ejecución
1.  Descargar el código: Obtén el archivo fuente conversor.c.
2.  Compilación: Abre la terminal en la carpeta del archivo y traduce el código a lenguaje de máquina con el siguiente comando:
gcc conversor.c -o conversor
3.  Ejecución: Una vez generado el archivo ejecutable, lánzalo con:
En Linux/Mac: ./conversor
En Windows: conversor.exe
4.  Interacción: Sigue el menú en pantalla para seleccionar el tipo de conversión e ingresar el monto deseado.
